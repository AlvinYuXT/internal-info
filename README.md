# internal-info

------

从北邮人论坛抓取的关于前端的实习招聘信息。



## 用法

1. 修改配置文件

   ```javascript
   {
    smtpConfig: {
      service: 'qq',
      auth: {
        user: 'example@qq.com',
        pass: 'pass'
      }
    },
    from: user,
    to: 'example@163.com',
    keyword: 'example',
    url: 'https://bbs.byr.cn/rss/board-ParttimeJob'
  }
   ```

2. node index.js
